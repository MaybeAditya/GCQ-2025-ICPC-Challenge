**Global CodeQuest 2025** was an advanced algorithmic programming contest following the **ICPC (International Collegiate Programming Contest)** scoring rules. The event brought together developers and students to solve complex problems focusing on:

* **Advanced Data Structures:** Segment Trees, Disjoint Set Union (DSU), Tries.
* **Graph Theory:** Network Flow, Shortest Path Algorithms.
* **Dynamic Programming:** Bitmask DP and Digit DP.
* **System Design Scenarios:** Load balancing simulation constraints.
